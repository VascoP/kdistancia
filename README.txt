Kdistancia
O Kdistancia e uma aplicacao para verificar a distancia de percursos feitos em passeios a pe, de bicicleta ou outros.
O objectivo da aplicacao sao os caminhos personalizados em situacoes que um dispositivo de captura GPS em tempo real nao esteve 
disponivel no treino, ou queremos so verificar o tamanho de determinado percurso antes de o fazermos.
Para percursos nao personalizados (ex: distancia da rua A a rua B) o proprio site do mapas do site ja faz um bom trabalho com estimativas
utilizando varios meios de transporte.
A aplicacao usa a API e os recursos dos mapas do sapo.

Kdistancia
The Kdistancia (how far) is an application to check the distance of costumized routes 
made in walks on foot, bicycle or other. 
The purpose of the application are the situations in that a GPS device for real-time capturing was not 
available in training, or so we can check the distance of a given route before actually going through it. 
For non customized routes (eg distance from street A to street B) mapas.sapo.pt's own 
service already does a good job with estimates using various means of transport. 
The application uses the API and features of mapas.sapo.pt
