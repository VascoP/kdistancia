<h1>QuantoCorri.com</h1>
<p>O QuantoCorri.com é uma aplicação para verificar a distância de percursos feitos em passeios a pé, de bicicleta ou outros.</p>
<p>O objectivo da aplicação são os caminhos personalizados em situações que um dispositivo de captura GPS em tempo real não esteve 
disponível no treino, ou queremos só verificar o tamanho de determinado percurso antes de o fazermos.</p>
<p>Para percursos não personalizados (ex: distância da rua A à rua B) o próprio site do mapas do site já faz um bom trabalho com estimativas
utilizando vários meios de transporte.</p>
<p>A aplicação usa a API e os recursos dos <a href="http://mapas.sapo.pt" target="_ablank">mapas do sapo</a>.
